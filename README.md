# Crispy
Crispy is a super basic terminal-based code editor designed to have typical keybinds as your everyday document editor. It was a project to practice writing C many years ago. It currently only supports Windows and is not actively maintained.

[![MIT License](https://img.shields.io/badge/License-MIT-green.svg)](https://choosealicense.com/licenses/mit/)


## Installment
To install this project build main.c using CMake
